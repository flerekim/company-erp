# 토양오염정화공사 ERP 시스템

## 프로젝트 개요
토양오염정화공사 전문 기업을 위한 ERP 시스템입니다. 수주 관리, 채권 관리, 프로젝트 관리 등의 기능을 제공합니다.

## 주요 기능
1. 수주 관리
   - 수주 등록/수정/삭제
   - 수주 현황 조회
   - 수주 상세 정보 관리
   - 첨부 파일 관리

2. 채권 관리
   - 채권 현황 조회
   - 연체 채권 관리
   - 수금 현황 관리

3. 대시보드
   - 주요 지표 현황
   - 최근 수주 현황
   - 연체 채권 알림

## 기술 스택
- Frontend: Next.js 14, React, TypeScript
- UI: Tailwind CSS, shadcn/ui
- Database: Supabase
- Authentication: Supabase Auth
